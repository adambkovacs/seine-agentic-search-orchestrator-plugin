The Council

Seven cognitive perspectives. None sees another's output. All challenge the same evidence.

Synthesizer Integration + Big-picture
Contrarian Stress-test + Prove it
Lateral Hunter Adjacent domains + Analogies
Source Critic Provenance + Authority + Bias
Pattern Spotter Trends + Correlations
